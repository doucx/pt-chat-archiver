# PonyTown 网页聊天记录存档器

一个功能强大的油猴脚本，旨在自动保存您在 PonyTown 中的聊天点滴。它不仅能将聊天记录安全地存储在您的浏览器本地，还提供了一个精致的 UI 供您随时查看、分析、导出和管理这些珍贵的记忆。

## ✨ 主要功能

-   **多服务器独立存档 (V6 架构)**：
    -   **自动识别**：脚本会自动检测您当前进入的服务器（如 Safe Server、18+ Server 等）。
    -   **独立空间**：每个服务器的聊天记录均独立存储，互不干扰。
    -   **只读查阅**：支持在当前服务器录制数据的同时，切换查看其他服务器的历史存档。
-   **全自动后台录制**：脚本在后台静默运行，利用 `MutationObserver` 技术实时抓取并保存聊天框中的每一条新消息。
-   **多维度频道分类**：
    -   自动识别并分类：`Local`、`Party`、`Whisper`（私聊）及系统消息。
    -   **合成频道**：自动生成 `Party-Local` 和 `Whisper-Local` 汇总频道，方便在 Local 视角下统一查看。
-   **交互式查阅体验**：
    -   **UI 刷新暂停**：通过 **⏸️/▶️** 按钮可随时冻结界面，方便您从容地阅读或勾选文字，而不会被源源不断的新消息顶走。
    -   **分页浏览**：支持超大容量存档的分页加载，确保在拥有数万条记录时依然保持极致的流畅度。
    -   **一键回正 (📍)**：在查看历史服务器存档时，点击图标可瞬间跳回当前正在录制的服务器。
-   **深度数据分析**：内置统计引擎，一键生成频道报告，分析最活跃的“话痨”用户以及聊天的峰值时间段分布。
-   **精准私聊识别**：通过设置您的“用户昵称”，脚本能智能判定私聊的方向（是您发送的还是接收到的），并以直观的图标标注。
-   **灵活的数据导出**：
    -   **纯文本复制**：快速复制当前频道的可读内容。
    -   **全量导出 (JSON)**：导出原始结构化数据，适合备份或开发者二次分析。
    -   **一键打包下载**：同时生成 `.txt` 和 `.json` 格式的压缩包，方便离线永久保存。

## ⚙️ 兼容性与要求

为了确保脚本能够精准解析历史消息的时间戳，您 **必须** 在 PonyTown 的游戏设置中完成以下配置：
-   **时间格式**：设置为 **24小时制**。
    -   路径：`Settings -> Display -> Time format display -> 24h`。

## 🔧 如何使用

1.  **安装脚本**：通过 Tampermonkey 等脚本管理器完成安装。
2.  **进入游戏**：脚本会自动检测游戏加载状态并静默启动。
3.  **管理界面**：点击屏幕**右下角**的悬浮图标 **📜** 即可呼出存档面板。
4.  **必要设置**：
    -   打开 **⚙️ 设置** 视图。
    -   在“用户昵称”处输入您当前的昵称，以启用私聊方向识别。
    -   在此处您可以切换“查看存档服务器”来翻阅不同服的历史记录。
5.  **日常操作**：
    -   在顶栏下拉框选择频道。
    -   点击 **📊 统计** 查看频道活跃度分析。
    -   点击文本框会自动进入“暂停模式”，方便您选择并复制文字。

## ⚠️ 重要提示

-   **本地隐私存储**：所有聊天记录均保存在您浏览器的 `localStorage` 中，**绝对不会**上传到任何第三方服务器。您的隐私完全掌握在自己手中。
-   **存储空间限制**：浏览器对本地存储有约 5MB 的硬性限制。脚本内置了空间占用监测，当空间告急时（超过 3.5MB）会显示黄色警告，请及时下载备份并进行“清空”或“清理重复记录”操作。
-   **录制限制**：由于技术限制，脚本仅能保存您当前在游戏内**激活切换到**的聊天频道（例如，若您一直待在 Local 频道而没点开过 Party 标签页，Party 频道的消息将无法被采集）。
-   **历史回溯限制**：脚本无法获取您安装它之前、或未在屏幕上渲染出来的旧聊天记录。
