# PonyTown 聊天记录存档器

这是一个专为 PonyTown 玩家设计的全自动、本地化聊天记录管理方案。它能静默、持久地记录您的对话点滴，并提供精致的交互界面，让这些珍贵的瞬间可查阅、可统计、可永久保存。

## ✨ 核心特性

### 1. 全自动无感录制
*   **即开即用**：脚本在后台静默运行，无需手动干预，自动抓取并保存聊天框中的每一条新消息。
*   **高可靠持久化**：采用多重保存机制，支持**自定义自动保存间隔**。即使意外断电或浏览器崩溃，您的数据也能安全留存在本地。
*   **断点检测**：智能识别由于刷新或切换标签导致的记录间断，并在历史记录中自动插入警告标记。

### 2. 极致的查阅与交互体验
*   **极简视觉设计**：全新进化的紧凑型 Header 布局，将操作高度符号化（Emoji 按钮），把最大化的屏幕空间留给文字本身。
*   **刷新暂停机制 (⏸️)**：阅读长篇记录时，点击暂停按钮可立即冻结 UI，防止新消息涌入导致的“滚动条跳动”。
*   **流畅分页系统**：即使累积了数万条聊天记录，强大的分页引擎也能确保查阅过程丝滑如初。
*   **一键复制 (📋)**：支持快速复制当前页面的纯文本内容，完美保留对话格式。

### 3. 多服务器智能隔离
*   **自动服务器识别**：系统会自动感知您当前进入的服务器（如 Safe Server, 18+ Server），并建立独立的存储空间。
*   **只读查阅模式**：允许您在录制当前服数据的同时，翻阅其他服务器的历史存档，互不干扰。
*   **一键回正 (📍)**：在翻阅历史时，点击图标可瞬间跳回当前正在录制的服务器视图。

### 4. 深度频道分类与分析
*   **多维度分类**：自动识别 `Local`、`Party`、`Whisper` 及系统消息。
*   **合成频道**：自动生成 `Party-Local` 与 `Whisper-Local` 汇总，方便您在 Local 视角下统一监控。
*   **聊天统计引擎 (📊)**：一键生成频道报告。谁是真正的“话痨”？聊天的峰值出现在什么时候？数据一目了然。
*   **私聊方向识别**：设置您的“用户昵称”后，系统能智能区分私聊的发送与接收状态。

### 5. 安全掌控与数据管理
*   **本地隐私至上**：所有数据均存储在浏览器的 `localStorage` 中，**绝不上传**到任何服务器。您的隐私，只有您能访问。
*   **多格式全量导出**：支持一键打包下载 `.json` (原始数据) 与 `.txt` (精排版文字) 备份。
*   **保存状态实时化**：设置界面实时显示“上次保存时间”，提供看得见的安全感。

---

## ⚙️ 关键配置项 (必读)

为了确保脚本能精准解析历史消息的时间戳，请务必在 PonyTown 游戏设置中完成以下配置：
*   **时间格式**：必须设置为 **24小时制**。
    *   *路径：Settings -> Display -> Time format display -> 24h*

---

## 🔧 使用指南

1.  **激活脚本**：安装后进入游戏，点击右下角的 **📜** 悬浮图标呼出面板。
2.  **初次设置**：
    *   点击 **⚙️** 进入设置页面。
    *   在“用户昵称”栏输入您的游戏昵称，以激活私聊方向识别。
    *   （可选）根据您的设备性能调整“自动保存间隔”与“分页大小”。
3.  **日常管理**：
    *   点击 **📋** 快速复制当前频道。
    *   定期在设置页面检查“存储占用”，若接近 5MB 限制，请及时使用“下载备份”并执行“清空”操作。

---

## ⚠️ 使用须知

*   **存储限制**：受浏览器 `localStorage` 的 5MB 限制，请留意界面的存储警告提示。
*   **录制范围**：脚本仅能录制您当前**已激活切换到**的聊天频道。若您从未点击过某个频道标签页，该频道的数据可能无法被完整采集。
