# PonyTown 聊天记录存档器

这是一个专为 PonyTown 玩家设计的全自动、本地化聊天记录管理方案。它能静默、持久地记录您的对话点滴，并提供精致的交互界面，让这些珍贵的瞬间可查阅、可统计、可永久保存。

## ✨ 核心特性

### 1. 海量数据存储
*   **IndexedDB 存储驱动**：采用 IndexedDB 数据库引擎，彻底突破浏览器传统存储的空间限制。您可以放心地长期保存数十万条聊天记录，无需担心空间耗尽。
*   **本地隐私至上**：所有数据均存储在您的本地数据库中，**绝不上传**到任何服务器。您的隐私，只有您能访问。

### 2. 极致的查阅与交互体验
*   **底部锁定模式**：在查看日志时，点击“末页”按钮或手动滑到底部，界面将自动锁定，新消息涌入时会实时推屏滚动。
*   **刷新暂停机制 (⏸️)**：阅读长篇记录时，点击暂停按钮可立即冻结 UI，防止新消息导致滚动条跳动。
*   **只读查阅视觉反馈**：当您查阅非当前录制服务器的存档时，界面会显示明显的“只读”标识与水印，确保操作清晰无误。
*   **分页系统**：高性能分页引擎，即使面对海量历史数据，查阅过程依然丝滑。

### 3. 多服务器智能隔离
*   **自动服务器识别**：系统会自动感知您当前所在的服务器（如 Safe Server, 18+ Server），并建立独立的存储空间。
*   **跟随服务器切换**：支持“自动跟随”功能，当您在游戏中切换服务器时，存档视图会自动跳转到对应的服务器记录。
*   **一键回正 (📍)**：在翻阅历史服务器存档时，点击定位图标可瞬间跳回当前正在录制的服务器视图。

### 4. 深度频道分类与分析
*   **多维度分类**：自动识别 `Local`、`Party`、`Whisper` 及系统消息。
*   **合成频道**：自动生成 `Party-Local` 与 `Whisper-Local` 汇总，方便在 Local 视角下统一监控。
*   **聊天统计引擎 (📊)**：一键生成频道报告，分析活跃度、发言峰值及社交统计。
*   **私聊方向识别**：设置用户昵称后，系统能智能区分私聊的发送与接收状态。

### 5. 数据安全与维护
*   **多格式全量导出**：支持打包下载 `.json` (原始数据) 与 `.txt` (精排版文字) 备份。
*   **智能去重清理**：针对短时间内大量刷屏或重复导入的异常记录，提供基于“爆发期”识别的清理算法。
*   **断点检测**：自动识别并标记由于刷新或切换标签导致的记录间断。

---

## ⚙️ 关键配置项 (必读)

为了确保脚本能精准解析历史消息的时间戳，请务必在 PonyTown 游戏设置中完成以下配置：
*   **时间格式**：必须设置为 **24小时制**。
    *   *路径：Settings -> Display -> Time format display -> 24h*

---

## 🔧 使用指南

1.  **激活脚本**：安装后进入游戏，点击右下角的 **📜** 悬浮图标呼出面板。
2.  **基本设置**：
    *   点击 **⚙️** 进入设置页面。
    *   在“用户昵称”栏输入您的游戏昵称，以激活私聊方向识别。
3.  **兼容性维护**：
    *   如果在系统升级后提示存在旧版 LocalStorage 备份，可在确认数据无误后点击“删除旧版备份”以释放浏览器冗余空间。
4.  **日常管理**：
    *   点击 **📋** 快速复制当前页面内容。
    *   查阅存档时，确保滚动条处于底端即可激活“自动跟随新消息”模式。

---

## ⚠️ 使用须知

*   **录制范围**：脚本仅能录制您当前**已激活切换到**的聊天频道。若您从未点击过某个频道标签页，该频道的数据可能无法被完整采集。
